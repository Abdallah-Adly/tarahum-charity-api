# Tarahum Charity API

Backend system for managing charity requests and donations.

## API Documentation

The project uses Swagger for API documentation.

- Local Swagger UI:  
  http://localhost:5555/api-docs

## Features
- Manage users (donors & beneficiaries)
- Handle requests
- Track donations
- API documentation with Swagger

## Tech Stack
- Node.js
- Express
- TypeScript
- Swagger

## Installation

```bash
npm install
